Reports

In the following section different reports can be produced. These reports can be viewed in two different ways: The browser's console window, or downloaded as a .csv file. Reports are initally run with default values, like in the case of 'Which customers had a transaction over $200', but if a new value is entered and the update button is clicked, then both the console report and the downloadable reports will be updated with the new requested value. The update button must be clicked after a value is entered before one of the save buttons is pressed or the report will still contain the old value. You can verify that the value has been updated by seeing the value change in the 'Which customers had a transaction over $____' line. If the dollar amount has changed, the new value is part of the console and savable reports.
